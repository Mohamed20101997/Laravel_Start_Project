# kaptanvip Task

## Running the Project

To run the project, use the following command:

```bash
php artisan serve
```

## Connecting to the Database

1. Create a database named `Task`.
2. Run the following command to set up the database schema and seed data:

```bash
php artisan migrate --seed
```
